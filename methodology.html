Methodology

Sources

All data scraped from Tripadvisor (painfully), and Instagram

Tools

To scrape data, I used webscraper.io with assistance from a BeautifulSoup Python script.

To clean the data, I used a number of Pandas Python scripts and Tableau Prep, as well as Texas A&M's geocoding service. Minor adjustments were made by hand, directly editing the CSV file, whenever it wasn't worth rewriting an entire workflow to catch a small bug.

Data visualization was done with Tableau Desktop, and the website was built by me using both vanilla HTML/CSS and Bootstrap.

Scripts/sitemaps/cleaned source data:

  • Cleaned, geocoded Tripadvisor data
  • Cleaned Instagram post data
  • Instagram scores per museum
  • Python script to get Tripadvisor links
  • Python script for scoring museums based on Instagram posts
  • An ill-fated, half-baked attempt to write my own geocoding script
  • Sitemap used to scrape Tripadvisor once links had been collected
  • Sitemap used to scrape Instagram for posts for a hashtag #note: webscraper.io has a limit of ~1000 results per scrape, but I knew I needed a larger dataset than 1000 across all museums to get any meaningful insights that weren't just noise. That being, I choose 15 museums that had at least 1000 posts with their hashtag, and with a sampling of both higher-count and lower-count hashtagged museums ran 15 of these sitemap scrapes, then combined the output files and proceeded.
